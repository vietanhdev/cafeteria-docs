# Contribution Guidelines

## How to contribute?

- **Repository:** [https://github.com/NextCafeteria/cafeteria](https://github.com/NextCafeteria/cafeteria).

Repositoy is open to all members of the community and we look forward to contributions from everyone. Follow the steps below to contribute to this project.

1. Fork the repository
2. Clone the repository to your local machine
3. Create a branch
4. Make necessary changes and commit those changes
5. Push changes to GitHub
6. Submit your changes for review

## Local Development

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) 18 or higher.

### Install and run

1. Clone the repository

```bash
git clone https://github.com/NextCafeteria/cafeteria
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

### Build the code

```bash
npm run build
```

## Coding style

- Use [Prettier](https://prettier.io/) for code formatting.
- Run the following command to format the code.

```bash
npm run pr
```
