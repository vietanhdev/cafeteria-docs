# Cafeteria

Online ordering and store management for coffe shops.

![](https://github.com/NextCafeteria/cafeteria/raw/master/docs/screenshot1.png)

- **Deployed at:** [https://cafenow.me](https://cafenow.me).
- **Frontend:** Next.js with Javascript, Tailwind CSS.
- **Database:** Google Firebase.
- **Authentication:** NextAuth.js with Google OAuth.
- **Deployment:** Vercel.
